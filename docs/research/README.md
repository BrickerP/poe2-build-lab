## Expert model distilled

A useful PoE2 site must model a build as a connected system:

`class/ascendancy -> passives -> skills/supports/spirit -> attributes -> gear slots/base types -> affixes/mod tiers -> defenses -> budget/trade -> patch validity -> in-game .build handoff`

The current static site covers only prose-level beginner concepts. It does not yet solve the real player jobs-to-be-done: "Can I equip/use this?", "What gear do I buy next?", "Is this build bait?", "What changed this patch?", and "Can I import this into the game?".
