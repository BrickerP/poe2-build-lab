# Finding 11: Reddit is best for pain discovery; X is best for update velocity

## Research result

Reddit and X serve different research roles.

- Reddit: best for durable user pain, build requests, gear confusion, trust gaps, and "what do I play?" demand.
- X: best for official update velocity, patch-note announcements, build-planner announcements, launch/live status, and creator hype signals.

## Evidence

Reddit evidence gathered from:

- r/pathofexile2builds homepage and build index
- build planner export discussions
- beginner/budget build questions
- attribute/stat-starvation complaints
- gear and defensive-layer questions
- endgame/progression pain surfaced through patch discussion

X evidence gathered from official Path of Exile posts surfaced in search results:

- 0.5.2 patch notes announcement
- Return of the Ancients patch notes announcement
- skill tree JSON availability
- Build Guide system announcement
- Return of the Ancients live announcement
- first two weeks recap signal

## Limitation

X pages are often hard to fully extract without logged-in/browser rendering. For source-backed claims, treat X as a directional/announcement source and resolve mechanics through official forum/docs whenever possible.

## Product implication

Research workflow:

1. Daily: check official patch notes and X official feed for new changes.
2. Daily: scan Reddit for repeated build/gear/attribute questions.
3. Weekly: promote repeated questions into long-tail guide pages.
4. Never base exact mechanics only on social snippets; confirm through official docs, game testing, wiki/PoE2DB, or patch notes.
