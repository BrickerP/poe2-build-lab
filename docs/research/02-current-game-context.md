# Finding 02: Current PoE2 context is 0.5 Return of the Ancients, not a stable solved meta

## Research result

Path of Exile 2 is currently in the 0.5 Return of the Ancients era. The current market is good for a guide/tool site because the patch introduced a large endgame reset, new mechanics, and ongoing hotfix churn. That creates search demand, but it also makes stale build content risky.

## Evidence

Official launch thread says Return of the Ancients is live and includes:

- new endgame storylines
- challenges and progression systems
- 15 bosses including four pinnacle encounters
- new crafting systems
- new Ascendancies: Martial Artist and Spirit Walker

Official 0.5.2 patch notes show ongoing endgame/mechanic changes after launch, including Atlas, Delirium, catalyst/minion modifier, controller map UI, and other changes.

X official account also amplified the update/patch cycle:

- 0.5.2 patch notes announcement
- Return of the Ancients patch notes announcement
- JSON availability for the Return of the Ancients skill tree
- Build Guide system announcement

## User pain

Players are not just asking "what is good?" They are asking "what is still valid after this patch and hotfix?" A static guide without patch labels becomes untrustworthy quickly.

## Product implication

Every build page needs:

- `patch_version`
- `last_reviewed`
- `known_broken_by_patch`
- `league/context`
- source links to official patch notes and creator build source
- a visible "not re-tested after patch" warning when needed
